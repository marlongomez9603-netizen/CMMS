### SEMANA 1 — 24 al 28 de marzo de 2026
**Tema: Fundamentos de la Planificación del Mantenimiento**

| Aspecto | Descripción |
|---|---|
| **Subtemas** | • Conceptos fundamentales de planificación del mantenimiento |
| | • Diferencia entre planificación y programación del mantenimiento |
| | • Objetivos de la planificación: disponibilidad, confiabilidad y mantenibilidad |
| | • Ciclo de gestión del mantenimiento (planificar, programar, ejecutar, controlar) |
| **Competencia** | Comprende la importancia de la planificación como eje central de la gestión del mantenimiento industrial |
| **Resultado de aprendizaje** | Identifica los elementos fundamentales del proceso de planificación y su rol en la optimización de recursos |
| **Estrategia de formación** | Clase magistral con apoyo audiovisual. Socialización de experiencias y contextualización con la realidad industrial |
| **Actividad presencial** | Exposición docente, mapa conceptual del ciclo de planificación, discusión grupal |
| **Trabajo independiente** | Lectura: Capítulo de planificación del mantenimiento (Dounce, E.). Elaborar mapa mental sobre los tipos y niveles de planificación del mantenimiento (4 horas T.I.) |
| **Horas presenciales** | 3 h 15 min |
| **Horas T.I.** | 4 h |

### SEMANA 3 — 6 al 10 de abril de 2026
**Tema: Mantenimiento Preventivo – Programación y Planes**

| Aspecto | Descripción |
|---|---|
| **Subtemas** | • Fundamentos del mantenimiento preventivo basado en tiempo y en condición |
| | • Diseño de planes de mantenimiento preventivo |
| | • Frecuencias de intervención y criterios de selección |
| | • Hojas de ruta y procedimientos estándar de mantenimiento |
| | • Listas de verificación (checklists) para inspecciones preventivas |
| **Competencia** | Diseña programas de mantenimiento preventivo considerando condiciones operativas y normativas industriales |
| **Resultado de aprendizaje** | Elabora planes de mantenimiento preventivo con frecuencias, procedimientos y recursos definidos |
| **Estrategia de formación** | Clase magistral con ejemplos prácticos. Taller de diseño de plan preventivo |
| **Actividad presencial** | Exposición docente, ejercicio práctico: diseño de un plan preventivo para un equipo electromecánico (motor eléctrico, bomba, compresor) |
| **Trabajo independiente** | Desarrollo de plan preventivo para un equipo asignado. Incluir: frecuencias, procedimientos, repuestos, herramientas y personal requerido (4 horas T.I.) |
| **Horas presenciales** | 3 h 15 min |
| **Horas T.I.** | 4 h |

### SEMANA 4 — 13 al 17 de abril de 2026
**Tema: Mantenimiento Correctivo – Programación y Órdenes de Trabajo**

| Aspecto | Descripción |
|---|---|
| **Subtemas** | • Programación del mantenimiento correctivo: priorización y gestión de emergencias |
| | • Órdenes de trabajo (OT): estructura, flujo y gestión documental |
| | • Backlog de mantenimiento: clasificación y priorización |
| | • Programación semanal y diaria de actividades de mantenimiento |
| | • Asignación de recursos: mano de obra, repuestos y herramientas |
| **Competencia** | Programa actividades de mantenimiento correctivo con criterios de priorización y gestión eficiente de recursos |
| **Resultado de aprendizaje** | Elabora órdenes de trabajo y programas de mantenimiento correctivo, priorizando intervenciones según criticidad |
| **Estrategia de formación** | Clase magistral. Taller práctico de elaboración de órdenes de trabajo y programación semanal |
| **Actividad presencial** | Exposición docente, taller: elaboración de OT y ejercicio de programación semanal con priorización de actividades |
| **Trabajo independiente** | Ejercicio: crear un programa semanal de mantenimiento con al menos 10 órdenes de trabajo priorizadas según criticidad (4 horas T.I.) |
| **Horas presenciales** | 3 h 15 min |
| **Horas T.I.** | 4 h |

### SEMANA 5 — 20 al 24 de abril de 2026
**Tema: Optimización de Recursos y Sistemas CMMS/GMAO**

| Aspecto | Descripción |
|---|---|
| **Subtemas** | • Optimización de recursos en mantenimiento: mano de obra, materiales y costos |
| | • Gestión de inventarios de repuestos y materiales |
| | • Introducción a los sistemas CMMS/GMAO (Computerized Maintenance Management System) |
| | • Funcionalidades principales: registro de activos, programación, historial, reportes |
| | • Demostración práctica de software de gestión del mantenimiento |
| **Competencia** | Implementa estrategias de optimización de recursos utilizando herramientas tecnológicas para la gestión del mantenimiento |
| **Resultado de aprendizaje** | Utiliza herramientas de software para la gestión y optimización de programas de mantenimiento |
| **Estrategia de formación** | Clase magistral con demostración de software. Ejercicio práctico guiado |
| **Actividad presencial** | Exposición docente, demostración de CMMS (IRCM/Excel), ejercicio práctico de registro de activos y programación de mantenimiento en software |
| **Trabajo independiente** | Ensayo: ventajas de implementar un sistema CMMS en una empresa industrial colombiana. Consulta en bases de datos (4 horas T.I.) |
| **Evaluación** | **Examen escrito – Parcial segundo corte** (programado por UNIPAZ – Semana de parciales) |
